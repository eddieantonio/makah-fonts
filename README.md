An application to test fonts' suitability to fully render the
entire Qʷi·qʷ·diččaq (Makah) alphabet.

Develop
-------

You must use `yarn`:

    yarn watch

Building
--------

You must first install dependencies with `yarn`:

    yarn install

Then you can run:

    yarn run rollup -c
